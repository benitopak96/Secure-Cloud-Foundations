## 🛡️ Project Overview
This repository serves as a production-ready, secure-by-default landing zone for Google Cloud Platform. 
Built using **Terraform**, this project demonstrates the integration of **DevSecOps** principles by 
automating security scanning at the pull-request level to prevent common cloud misconfigurations.
